# Neuroity – AI Dataset Sources & Programmatic Access

## Overview

This repository contains my submission for the **Neuroity Generative AI Engineering Internship** assignment on researching AI dataset platforms and demonstrating programmatic dataset access.

The project evaluates **12 major AI dataset platforms**, demonstrates programmatic dataset search and access through Python, compares each platform's capabilities, and proposes an automated data collection pipeline for AI and machine learning workflows.
