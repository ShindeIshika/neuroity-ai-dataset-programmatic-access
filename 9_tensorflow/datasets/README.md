# TensorFlow Datasets – Platform Note

## Status
**Partially Functional**

## Compatibility Notice

TensorFlow Datasets (TFDS) currently has limited compatibility with **Python 3.14.x** because the TensorFlow framework has not yet released official support for this Python version.

## Current Environment

- **Python Version:** 3.14.3
- **TFDS Installation:** Successful
- **Dataset Discovery:** Supported
- **Dataset Download & Loading:** Requires a compatible TensorFlow installation

## Recommendation

For full TensorFlow Datasets functionality, use one of the following environments:

- Python **3.12 or earlier**
- Google Colab
- A virtual environment with a TensorFlow-supported Python version

## Assignment Note

The platform has been thoroughly researched and documented as part of this assignment. The accompanying script demonstrates the intended programmatic workflow, while full dataset download functionality depends on TensorFlow compatibility with the Python environment.

**Note:** This limitation is due to the current TensorFlow release cycle and is not specific to this implementation.
